## Instalação

```bash
$ python -m venv venv
# Instalar um ambiente virtual
```

```bash
$ source venv/Scripts/activate
# Acessar o ambiente virtual
# Obs.: Terminal Bash
```

```bash
$ pip install -r requirements.txt
# Instalação de dependências
```

## Executando a aplicação

```bash
# Rodar o projeto
$ python Escala.py
```
